You Got Options!

Sometimes you are faced with making a decision between multiple options and you need to mull it over to decide which choice is the most optimal. You Got Options is an app that provides users with the ability to weigh all their options and visually compare all factors involved using a point system. You Got Options can help you think through any decision, large or small.

You Got Options provides users with the ability to add decisions and add multiple options for each decision. Each option can contain multiple factors where users can assign a point value for each factor and display the total points for each option, allowing them to visually compare each option to make an informed decision.

My Role: This was an independent project. Creating the entire app included planning out and setting up the database that includes three levels of a nested hierarchy of decisions, options, and factors. I created all the apps functionality, designed the logo, and styled all pages to be mobile responsive.

Technology Used: JavaScript (Node.js & Express,), Sequelize, React, Redux, HTML, CSS, React Bootstrap
